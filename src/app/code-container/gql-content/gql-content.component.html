1. Get a token

First of all, yous should get a token for your application by mutation `createToken` 

    mutation {{'{'}}
        createToken
    {{'}'}}
  
    {{'{'}}
        "data": {{'{'}}
            "createToken": "your token"
        {{'}'}}
    {{'}'}}

2. Add URL of website

Should add an URL of website by `addUrl` mutation. Previewly will return a pre-flight info about site preview and start grab the image. 

    mutation {{'{'}}
        addUrl (
          token: "your token",
          url: "https://www.google.com/"
        ) {{'{'}}
            id
            image
            status
            url
        {{'}'}}
    {{'}'}}
  
    {{'{'}}
        "data": {{'{'}}
            "addUrl": {{'{'}}
                "id": 1,
                "image": "https://previewly.top/assets/loader-200px-200px.gif",
                "status": "pending",
                "url": "https://www.google.com/",
            {{'}'}}
        {{'}'}}
    {{'}'}}
